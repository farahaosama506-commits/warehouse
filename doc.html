📦 نظام إدارة المستودعات WMS

نظام متكامل لإدارة المستودعات والمخزون والحركات مع تقارير متقدمة وإشعارات ذكية

📋 نظرة عامة

نظام إدارة مستودعات (WMS) مبني باستخدام Next.js 16 + TypeScript + Supabase. يوفر إدارة كاملة للمنتجات والمستودعات والأوامر مع نظام صلاحيات متقدم وتقارير تحليلية وتصدير PDF.

👥 الأدوار الوظيفية

الدورالصلاحيات
مدير عامكل الصلاحيات - وصول كامل للنظام
مشرف مستودعالمنتجات، الأوامر، المستودعات، التقارير (عرض فقط)
عامل مستودععرض المنتجات، إنشاء أوامر فقط
مدير نظامالمنتجات، المستودعات، المستخدمين، الإعدادات
مستثمرقراءة فقط - التقارير والمستودعات

🚀 التثبيت والتشغيل

المتطلبات

  • Node.js 18+
  • حساب Supabase

خطوات التثبيت

git clone https://github.com/farahaosama506-commits/warehouse.git
cd warehouse
npm install

إعداد قاعدة البيانات

  1. أنشئ مشروع في Supabase
  2. اذهب إلى SQL Editor وشغّل ملف schema.sql
  3. أنشئ ملف .env.local:
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
SUPABASE_SERVICE_ROLE_KEY=your-service-role-key

التشغيل

npm run dev

افتح http://localhost:3000

✨ المميزات

📦 إدارة المنتجات

  • إضافة، تعديل، حذف المنتجات
  • باركود للوصول السريع
  • تصنيفات وفئات
  • تتبع تواريخ الصلاحية
  • تحديد الحد الأدنى والأقصى للمخزون
  • حالات المنتج: نشط، غير نشط، نافذ المخزون، متوقف

🏗️ المستودعات والمواقع

  • إدارة مستودعات متعددة
  • أنواع تخزين: جاف، مبرد، تجميد، مواد خطرة
  • مواقع تخزين هرمية: قسم ← رف ← عمود ← خلية
  • حساب السعة المستخدمة تلقائياً من المنتجات

📋 الأوامر والحركات

  • أمر استلام - إضافة مخزون
  • أمر تجهيز - صرف مخزون
  • نقل داخلي - نقل بين المستودعات
  • بنود متعددة لكل أمر
  • تحديث المخزون تلقائياً
  • سجل الحركات اليومية

🔔 الإشعارات الذكية

  • تنبيهات المخزون المنخفض (حسب النسبة المحددة في الإعدادات)
  • تنبيهات اقتراب الصلاحية (حسب عدد الأيام المحددة)
  • تحليل سلوك المستخدمين (محاولات دخول مشبوهة، حذف جماعي)
  • اكتشاف الأخطاء المتكررة في العمليات
  • تقارير أسبوعية تلقائية
  • تفعيل/تعطيل كل ميزة بشكل منفصل
