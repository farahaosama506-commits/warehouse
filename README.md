## ✨ المميزات

### 🔐 نظام المصادقة والصلاحيات
- تسجيل دخول كامل مع Rate Limiting
- 5 أدوار: مدير عام، مشرف مستودع، عامل مستودع، مدير نظام، مستثمر
- صلاحيات متقدمة لكل دور (عرض، إنشاء، تعديل، حذف)
- حماية الصفحات تلقائياً حسب الصلاحيات

### 📦 إدارة المنتجات
- CRUD كامل مع باركود وتصنيفات
- تحديد الحد الأدنى والأقصى للمخزون
- تتبع تواريخ الصلاحية
- ربط مع المستودعات ومواقع التخزين

### 🏗️ المستودعات والمواقع
- إدارة مستودعات متعددة
- أنواع تخزين: جاف، مبرد، تجميد، مواد خطرة
- مواقع تخزين: قسم ← رف ← عمود ← خلية
- حساب السعة المستخدمة تلقائياً

### 📋 الأوامر والحركات
- أوامر استلام، تجهيز، نقل داخلي
- بنود متعددة لكل أمر
- تحديث المخزون تلقائياً
- سجل الحركات اليومية

### 📊 التقارير والتحليلات (8 تقارير)
- **قيمة المخزون** - الأصول المجمدة
- **سرعة البيع** - الأكثر مبيعاً مقابل الراكد
- **إنتاجية الموظفين** - عدد الأوامر المنجزة
- **المهام المعلقة** - أوامر لم تكتمل
- **سجل التدقيق** - من قام بأي عملية ومتى
- **صلاحية المنتجات** - تنبيهات قبل الانتهاء
- **التكاليف اللوجستية** - نقل، تخزين، استلام
- **تقرير الخسائر** - تالف ومنتهي الصلاحية
- تصدير PDF لجميع التقارير

### 👥 العملاء والموردين
- إدارة جهات الاتصال
- تصنيف: عميل، مورد، كلاهما
- كشف حساب مع الرصيد

### 🔔 الإشعارات الذكية
- تنبيهات المخزون المنخفض
- تنبيهات اقتراب الصلاحية
- تحليل سلوك المستخدمين
- اكتشاف الأنماط غير الاعتيادية
- تقارير أسبوعية تلقائية

### ⚙️ الإعدادات
- وحدات القياس مع معاملات التحويل
- إعدادات النظام العامة
- تفعيل/تعطيل الإشعارات الذكية
- إعدادات الأرشفة

### 🛡️ الأمان
- Rate Limiting على API
- Zod Validation على جميع المدخلات
- حماية ضد هجمات القوة العمياء
- سجل تدقيق كامل

### 🎨 واجهة المستخدم
- تصميم Minimal Premium
- دعم كامل للغة العربية RTL
- Responsive Design (موبايل، تابلت، ديسكتوب)
- Toast Notifications
- Confirm Dialogs
- Animations سلسة
